- **X Image:** 1200x675 px

- **Representation:** Diverse family compositions
- **Seasonal Context:** Colorado winter accuracy

## Technical Implementation Notes

### For All Images:
- **Overlay Specifications:** Dark Slate (#2C3E50) semi-transparent bar at 80% opacity
- **Overlay Positioning:** Consistent bottom third placement across all images for brand consistency
- **Text Spacing:** Minimum 20px margin from overlay bar edges; headlines 48-56px; subtitles 28-32px
- **Stock Photo Sources:** Prioritize Unsplash and Pexels for authentic, diverse imagery
- **Quality Standards:** Minimum 1200px width; professional lighting; sharp focus on subject
- **Diversity Requirements:** Include varied family structures, ethnicities, abilities across story set
- **Colorado Context:** Include seasonal/landscape elements appropriate to February context

### File Naming Convention:
- `COS-Parenting-Story[1-7]-X-[YYYYMMDD].jpg`
- `COS-Parenting-Story[1-7]-Facebook-[YYYYMMDD].jpg`
- Example: `COS-Parenting-Story1-X-20260222.jpg`

### Color Hex Values for Overlay Implementation:
- **Overlay Bar:** #2C3E50 @ 80% opacity (RGBA: 44, 62, 80, 0.8)
- **Headline White:** #FFFFFF
- **Headline Teal:** #0097A7
- **Subtitle Gold:** #F4C542
- **Subtitle Coral:** #FF6F61
- **Subtitle White:** #FFFFFF
